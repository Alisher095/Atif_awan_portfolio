# Atif Awan Portfolio

A clean, fast portfolio website for a backend-focused Python developer. Built with HTML, CSS, and JavaScript and designed for GitHub Pages.

## Structure

- `index.html` — Home page
- `projects.html` — Projects listing
- `assets/css/style.css` — Styles
- `assets/js/main.js` — Scripts
- `components/` — Shared navbar and footer

## Customize

1. Update your email, GitHub, and LinkedIn links in `index.html` and `projects.html`.
2. Replace project placeholders with your real repositories and live API links.
3. Add screenshots inside `assets/images/` if desired.

## Deploy (GitHub Pages)

1. Create a repository named `atifawan.github.io`.
2. Push this project to the main branch.
3. In GitHub: Settings → Pages → Source → `main`.
4. Your site will be live at https://atifawan.github.io.

## Custom Domain (GitHub Student Pack)

1. Purchase a domain from a Student Pack provider.
2. In GitHub: Settings → Pages → Custom domain.
3. Update DNS records as shown by GitHub.
